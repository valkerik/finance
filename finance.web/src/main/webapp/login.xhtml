Описание авторизации:

  • Authentication: когда вы входите в систему, ваши учетные данные проверяются по базе данных на соответствие. Если совпадение найдено вам разрешено продолжить (вы прошли аутентификацию); если нет, получите доступ отказано.
  • Authorization: Как только вы пройдете аутентификацию, вы должны быть авторизованным для доступа к определенному контенту. Например, общий пользователь может быть не авторизован для доступа к разделам веб-сайта, предназначенным только для администраторов. Авторизация основана на идее, что учетные записи пользователей принадлежат к одной или нескольким группам или роли. Это членство приводит к получению разрешения на просмотр определенных части веб-сайта.
